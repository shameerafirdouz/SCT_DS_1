Task 1: Data Visualization

📌 Objective

The goal of this task is to create basic visualizations using a dataset.
This helps in understanding data distribution and patterns.

---

📊 Dataset Used

- Titanic Dataset (CSV file)
- Contains information like passenger age, gender, survival, etc.

---

🛠 Tools & Libraries

- Python
- Pandas
- Matplotlib
- Seaborn

---

📈 Visualizations Created

1. Bar Chart

- Shows distribution of passengers based on gender
- Helps compare number of male and female passengers

2. Histogram

- Shows distribution of passenger ages
- Helps understand how ages are spread in dataset

---

▶️ How to Run

1. Install required libraries:

pip install pandas matplotlib seaborn

2. Run the Python file:

python task1.py
